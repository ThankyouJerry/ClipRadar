Scoring Model

지표 기준

종합 하이라이트 점수

클립 데이터가 있으면 채팅 급증도 40% + 클립 조회 영향 25% + 클립 생성 밀도 20% + 키워드 급증도 15%

현재 채팅 기반 분석에서는 채팅 급증도 70% + 키워드 급증도 30%로 계산합니다.

채팅 급증도

구간 분당 채팅량을 해당 스트리머의 평소 분당 채팅량과 비교한 배수

클립 영향도

구간 주변 클립 생성 밀도와 조회수 확산을 분리 계산한 뒤 합산

추천 컷 범위

채팅이 실제로 튄 핵심 반응 구간 앞 30초와 뒤 60초를 붙여 편집자가 바로 확인할 수 있는 범위

하이라이트 타입

채팅 키워드와 반응 문맥으로 웃긴 리액션, 고점 리액션, 놀람/사건, 채팅 급증으로 우선 분류

타입 신뢰도

분류 타입과 맞는 키워드 비중을 계산해 하이라이트 타입 옆에 0-100% 기준으로 표시합니다.

대표 채팅 샘플

해당 구간에서 실제로 수집된 채팅 일부를 보여줘 편집자가 반응 맥락을 빠르게 확인할 수 있게 합니다.

인접 구간 병합

같은 VOD 안에서 90초 이내로 이어지는 30초 후보는 최대 3분까지 하나의 연속 반응 구간으로 합쳐 중복 카드를 줄입니다.

고정/제외

좋은 후보는 고정해 위로 올리고, 애매한 후보는 제외해 ClipCatcher용 JSON/CSV/HTML 내보내기에서 자동으로 빼줍니다.

Confidence Guide

레이더 신뢰도란?

레이더 신뢰도는 ClipRadar가 뽑은 하이라이트 후보에서 서로 다른 신호가 얼마나 같은 구간을 가리키는지 나타내는 값입니다. 숫자가 높을수록 채팅, 클립, 키워드가 같은 순간에 함께 반응했다는 뜻입니다.

채팅 신호

평소 대비 채팅량이 튄 구간인지 확인합니다.

클립 신호

해당 구간 주변에 클립 생성 또는 조회 확산이 있는지 봅니다.

키워드 신호

웃음, 놀람, 특정 밈/상황어가 짧은 시간에 몰렸는지 봅니다.

해석 기준

90% 이상은 강한 후보, 75-89%는 좋은 후보, 60-74%는 검토 후보입니다.

Export Guide

ClipCatcher용 JSON은 어디에 쓰나요?

ClipCatcher용 JSON은 ClipRadar가 찾은 하이라이트 구간을 ClipCatcher에서 바로 가져오기 위한 데이터 파일입니다. 고정 후보는 먼저 정렬하고, 제외 후보는 자동으로 빼서 저장합니다.

ClipCatcher 연동

시작/종료 시간을 읽어 하이라이트 구간만 자동 다운로드할 수 있습니다.

편집자 작업표

TOP 구간, 점수, 선정 이유를 편집자가 확인할 목록으로 변환할 수 있습니다.

리포트 자동 생성

주간 HTML 리포트나 블로그 글을 같은 데이터로 다시 만들 수 있습니다.
