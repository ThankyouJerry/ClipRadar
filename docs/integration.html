Integration

ClipCatcher와 함께 쓰는 구조

ClipRadar는 하이라이트 후보를 찾고, ClipCatcher는 실제 VOD 구간 다운로드를 담당합니다. 둘을 분리하면 분석과 다운로드를 각각 안정적으로 개선할 수 있습니다.

ClipRadar가 만드는 것

VOD URL, 추천 컷 시작/종료 시간, 하이라이트 타입, 근거 키워드, 대표 채팅 샘플이 들어 있는 JSON 파일입니다.

ClipCatcher가 쓰는 것

cutStartTimeSeconds와 cutEndTimeSeconds를 읽어 해당 구간만 다운로드합니다.

Setup

GitHub Pages 구성

이 사이트는 정적 파일만 사용하므로 GitHub Pages에서 무료로 호스팅할 수 있습니다. 저장소 설정에서 Pages 소스를 `main` 브랜치의 `/docs` 폴더로 지정하면 됩니다. 실시간 VOD 분석은 로컬 앱 또는 별도 서버가 필요합니다. 

Repository Settings
-> Pages
-> Build and deployment: Deploy from a branch
-> Branch: main
-> Folder: /docs

Download Copy

현재 배포 방식

ClipRadar는 아직 별도 실행 파일 Release를 제공하지 않는 로컬 MVP입니다. 저장소를 받은 뒤 python3 server.py로 실행하고, 분석 결과는 ClipCatcher용 JSON, CSV 작업표, HTML 리포트로 저장할 수 있습니다.

실행 파일 배포가 준비되면 GitHub Release와 이 페이지를 함께 업데이트하면 됩니다.
