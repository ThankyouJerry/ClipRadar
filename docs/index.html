ClipRadar

긴 VOD에서
편집할 순간만 빠르게 찾습니다.

ClipRadar는 치지직 VOD의 채팅 반응과 키워드 급증을 분석해 하이라이트 후보를 만들고, ClipCatcher에서 바로 쓸 수 있는 구간 JSON과 편집자용 작업표를 내보냅니다.

실제 리포트 보기 ClipCatcher 연동 보기
30초 채팅 반응을 기본 30초 단위로 나눠 후보를 계산합니다.
+30/+60 핵심 반응 앞 30초, 뒤 60초를 붙여 추천 컷 범위를 만듭니다.
JSON ClipCatcher가 읽을 수 있는 다운로드 구간 데이터로 저장합니다.

What It Does

편집자가 바로 판단할 수 있게

채팅 급증 감지

VOD 전체 기준보다 채팅량이 크게 튄 구간을 먼저 찾습니다.

하이라이트 타입 분류

웃긴 리액션, 고점 리액션, 놀람/사건, 채팅 급증으로 후보를 분류합니다.

작업 파일 내보내기

ClipCatcher용 JSON, CSV 작업표, HTML 리포트로 후보를 재사용합니다.

Current Scope

현재 MVP 분석 범위

실제 VOD 분석은 치지직 다시보기 채팅과 반응 키워드를 기준으로 동작합니다. 샘플 리포트에는 클립 지표가 함께 들어 있지만, 실제 클립 데이터 연결은 다음 단계에서 보강할 예정입니다. GitHub Pages는 소개와 문서 공유용이며, 새 VOD 분석은 로컬에서 python3 server.py를 실행해 사용합니다.

ClipRadar MVP는 로컬 분석 도구이며, 이 페이지는 GitHub Pages용 소개 사이트입니다.
